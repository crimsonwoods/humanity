humanity
================

Summary
----

C++ class library for Android native code development.

humanityはAndroidのネイティブコード開発用に作られたC++のクラスライブラリです。


Build
----

    $ git clone https://github.com/crimsonwoods/humanity.git
    $ cd humanity/android
    $ ndk-build

Documents
----

    $ cd humanity/docs
    $ ./makedocs

documents are written in Japanese.

ドキュメントは日本語で書かれています。
